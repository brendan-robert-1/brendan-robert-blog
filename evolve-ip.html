Evolve IP: Automated Invoicing and Billing

Evolve IP has a large B2B customer base with custom contracts regarding bundles, seats packages, and discounts. Becaue of this generating monthly Invoices for each customer was very error prone and included many manual processes. This included manually checking database values to make sure the stated provisoned "bundles" matched up with the underlying components of the bundle

I wrote primarily Java Spring CronJobs that would run each night to reconcile any differences between provisioned services and custom bundles. This involved normalizing the SQL data model with regards to customer bundles, executing external rest calls to check third party service usages, and adjusting billing tables. Lastly, reports were generated and distributed to the sales, invoicing, and billing teams alerting them of any large changes as a result of nightly reconcilliation.

This completly automated my Billing Analyst job at the time as well as two others with my same title. It also allowed us to greatly reduce the number of "credit disputes" that our customers would submit for our previous inaccuracies with our old billing system. It also greatly improved the customer relationship.

Main tools used: Java, Spring, Cron, Linux, SQL, REST
